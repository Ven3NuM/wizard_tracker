# Wizard Tracker

A React-based web application for tracking scores in the Wizard card game.

## Features

- Support for 3-6 players
- Dynamic round counts based on player count:
  - 3 players = 20 rounds
  - 4 players = 15 rounds
  - 5 players = 10 rounds
  - 6 players = 5 rounds

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

## Tech Stack

- React 18.3
- Vite 6
- ESLint
